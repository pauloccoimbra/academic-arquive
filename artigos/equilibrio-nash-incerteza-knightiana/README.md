# Equilíbrio de Nash sob incerteza Knightiana

**Status:** em andamento
**Linha de pesquisa:** teoria dos jogos computacional
**Venue-alvo:** resumo estendido — IWGTEA 2026; artigo de software — SoftwareX/JOSS

## Resumo

Framework computacional (NashUncertainty) para cálculo de equilíbrios de Nash em jogos onde os jogadores enfrentam incerteza Knightiana (não-probabilística) sobre os payoffs, ao invés do tratamento bayesiano clássico. Inclui aplicação com GUI e suíte de testes.
