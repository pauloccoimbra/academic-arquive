# Arquivo Acadêmico — Paulo C. Coimbra

Repositório de artigos, notas de aula e materiais de curso e extensão. Serve como o acervo de conteúdo por trás do [site acadêmico](#) (hub), que apenas referencia e navega o que está aqui.

Software e aplicações completas (com código-fonte próprio e releases) vivem em repositórios dedicados — ver [`docs/repositorios-relacionados.md`](docs/repositorios-relacionados.md). Aqui ficam **documentos**: PDFs, LaTeX, slides, notas, exercícios.

## Estrutura

```
academic-archive/
├── artigos/            # papers e working papers
├── livros/              # textos didáticos de autoria própria
├── notas-de-aula/       # material de cada disciplina lecionada
├── extensao/             # material dos projetos de extensão
└── docs/                 # convenções e guias de uso deste repositório
```
